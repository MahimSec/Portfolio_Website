About Me

I am a SOC Analyst and Blue Team defender focused on threat detection, incident response, digital forensics, and SIEM operations. I work with open-source security tools, especially Wazuh, to identify attacks, investigate malicious activity, and strengthen defensive security.

I enjoy breaking down how threats work, analyzing malware behavior, and using AI-driven automation to improve SOC workflows. My interests include threat hunting, security monitoring, network forensics, and building practical defensive solutions.

I am continuously learning, practicing, and improving my skills to grow into a strong Security Engineer. Cybersecurity is not just my career — it is my passion, and I aim to contribute to a safer and more secure digital world.

Core Skills & Competencies

SOC Operations

  • Incident Triage: Expert handling of L1 incidents from detection to containment.
  • Alert Analysis: Correlating cross-source logs (Wazuh, Suricata) to identify true positives.
  • Performance Metrics: Focused on reducing MTTD/MTTR through efficient workflow optimization.

Threat Hunting

  • Hypothesis-Driven: Proactively searching for undetected threats using MITRE ATT&CK TTPs.
  • Endpoint Forensics: Leveraging Sysmon and Wazuh agent telemetry for deep-dive investigations.
  • Query Development: Crafting advanced KQL/Lucene queries to uncover anomalous patterns.

Automation & Playbooks

  • Security Automation: Python scripting for automated IOC enrichment (VirusTotal) and response.
  • Infrastructure: Automating SIEM deployment and agent provisioning using Bash/Docker.
  • Custom Tooling: Building custom security tools and parsers to bridge integration gaps.

ML-based Anomaly Detection

  • Behavioral Analytics: Implementing statistical baselines to detect anomalies in network/user behavior.
  • False Positive Reduction: Applying ML techniques to filter high-volume alert streams.
  • Traffic Analysis: Analyzing flow data (Zeek) to spot deviations indicative of C2 or exfiltration.

SIEM Engineering

  • Architecture Design: Deploying and managing scalable Elastic Stack and Wazuh environments.
  • Rule Development: Writing and testing custom detection rules (Sigma/YARA) for evolving threats.
  • Log Pipeline: Configuring Logstash/Filebeat for efficient parsing and normalization.
